This folder contains documentation and test cases derived from the Chipmunk2D implementation.  The content of the folders are as follows:

Chipmunk_UnitTesting	- Cunit tests for Chipmunk using Visual C++
Code Changes		- modifications to the private header files of Chipmunk
			to improve the separation of concerns
Documentation (MIG)	- automatically generated documentation using doxygen
Game physics lib….pdf	- problem statement
MG			- module guide (high level design)
Physics_Game_Lib….bib	- BibTeX file with references
Readme.txt		- this file
SRS			- software requirements specification for Chipmunk2D
V&V			- Verification and validation plan for Chipmunk2D
